# Proje Özeti:  
Bu projede, insanların uyku kalitesi analiz edilmiştir. Uyku kalitesindeki en büyük etkinin uyku süresi olduğu ve onun arkasından stres seviyesinin etkili olduğu ortaya konmuştur.


# Kullanılan veri:
[Health and Sleep relation 2024](https://data.mendeley.com/datasets/46j8wrc7p7/1)
